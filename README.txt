ONLINE PRICING TOOLS

I wrote this to scrape pricing from NZ online retail stores.
This saves time from going to each site to look up pricing.
It will take that information and enter it into an excel spreadsheet.

If used inside a company network, add the proxy details in the top of the script.

Written in Python using the following libraries:
- URLlib
- BeautifulSoup
- XLSXWriter
- DateTime
- SSL
- OS
